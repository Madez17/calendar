# Front-End

## Environment settings

1. Run the following commands:

  - Install dependencies:
      ```
      npm install
      ```

  - Start running the server:
      ```
      npm start
      ```
## Unit Testing
- To run test:
      ```
      npm run test
      ```

## Dependencies

 ```
  - node-sass
  - moment
  - react
  - react-date-picker
  - redux
  - react-redux
  - redux-devtools-extension
  - prop-types
```
